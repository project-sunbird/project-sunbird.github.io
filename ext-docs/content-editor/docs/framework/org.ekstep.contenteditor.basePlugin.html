Class: basePlugin

basePlugin

new basePlugin()

The base plugin class that all editor plugins inherit from. It provides the common support contract for all plugins. Plugins can override specific methods to change the behavior. The most common scenario would be to override the implementation of fabric callback methods to detect interactivity on the canvas.

Lesson plan is a structured outline of a given topic/unit/chapter to be taught in any given day. Lesson plan aids to streamline teaching and create an engaging learning experience.

Prerequisites

Step Screen
1. You are logged in
2. You are currently on My Workspace page
3. You want to create a lesson plan
4. You have clicked the Lesson Plan tile

Creating Lesson Plan

Step Screen
1. Enter a Name for the lesson plan
2. Select other metadata from their respective dropdowns:
a) Curriculum
b) Medium
c) Subject
d) Class
3. Click Start Creating to create the lesson plan

Adding Metadata

Step Screen
1. Enter Title
2. Enter Description for the lesson plan
3. Select the following from their respective drop-down lists:
a) Curriculum
b) Subject
c) Grade
d) Medium
e) Language
f) Concepts
4. Enter Notes to Teacher, if any
5. Enter Duration of the lesson plan
6. Click Update to update the metadata
7. Click Lesson Plan on the left navigation pane to add other units of content to the plan

Adding Image to Lesson Plan metadata

Step Screen
Click the Image icon to add an image
1. You can add images either by selecting an image from:
  • A list or uploading images
  • Your local machine to use immediately

    • 2. Click My images to view a list of all image files that you have previously, uploaded
    3. Click All image to view a list of image files from the repository
    Note: The repository contains all image files added and uploaded by any users
    5. Choose an appropriate image file from any list and click Select
    6. Click Upload and use to upload and use images from your gallery
    1. Choose File to browse and click Select a file from your local drive or open the explorer in a separate window or drag and drop the required file on the screen
    Note: Supported file formats are: .jpeg, .jpg, .png
    2. Under Copyright and License, select if you want to:
    a) Make it available to everyone
    b) Not share with anyone
    4. Add image details in the Asset Caption area
    5. Click Upload and Use to upload the image

    Adding Content to Lesson Plan

    Step Screen
    1. Click the + icon on the left navigation pane to add:
    a) Lesson Plan Unit
    b) Collection
    c) Resource
    d) Story
    e) Worksheet

    Adding Lesson Plan Unit

    Step Screen
    1. Enter Unit Title
    2. Enter Description for the unit
    3. Enter Notes to Teacher, if any
    4. Click the Image icon to add an image to the lesson plan unit
    5. Click Update to save the metadata
    6. Click Save to save the lesson plan unit

    Adding Collections, Resources, Stories and Worksheets

    Step Screen
    1. On clicking Collections, Resources, Stories or Worksheets the Add Resource page opens
    2. Type keywords to search for content
    3. For faster and effective search of the lessons, you can apply filters. To apply filters, select from the drop-down lists available for Language, Grade, Content Type, and Concept
    4. Click Apply. Results are loaded on the right pane
    5. Select the content you want to add and click Add to add the selected content

    Sending Lesson Plan for Review

    Before the Lesson Plan is publihed, it must be reviewed. You can send it for reviewing and publishing.

    Step Screen
    1. Click Save on the header to save the lesson
    2. Click Send for review to send the created lesson for review
    3. Click Limited sharing to publish the lesson to the limited audiences

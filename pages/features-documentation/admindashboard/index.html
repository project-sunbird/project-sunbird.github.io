The Administrator Dashboard provides the administrator a unified view of the organizationâ€™s content creation and consumption patterns. The data collected from multiple components is integrated and graphically represented. This helps to gauge and assess the value, popularity and acceptance of content created and used on the portal. The dashboard shows views of content created, reviewed, published, consumed etc. Administrators can also download the data as a .csv file.

Prerequisite

Step Screen
1. You are logged in as the administrator
2. You are currently on Home page. You want to view, review and analyze content creation or consumption patterns
3. You have clicked Admin Dashboard from the profile drop down

Selecting Organization

Step Screen
1.Click Select Organization from the drop down, to select the organization for which you require the analytics

Creation Analytics

Step Screen
1. Select Creation from the drop down
2. Click and select the duration for which you want to view data. You can view and analyse the following quantitative and qualitative data for the selected timeframe. You can view:
a) Total number of contents created
b) Number of authors
c) Number of reviewers
d) Total content created on the portal
e) Total content sent for review
f) Total content published
3. Click CSV to download the view as a report in .csv format. The report is sent to your registered e-mail ID

Consumption Analytics

Step Screen
1. Select Consumption from the drop down
2. Click and select the duration for which you want to view data about the number and frequency of the users visiting the portal to consume the content. The consumption graph summarizes the following quantitative and qualitative data:
a) Number of user visits
b) Content consumption time viz. the time spent by users consuming content on the portal
c) Average time spent by user per visit
3. Click CSV to download the view as a report in .csv format. The report is sent to your registered e-mail ID

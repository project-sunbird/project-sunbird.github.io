baseLauncher.

new baseLauncher.()

Base Launcher to launch any type launcher i.e Ecml/Html/Pdf/Youtube/video/h5p launcher. child launcher can overide the specific method to change the behavior. To extend the functionality of the baseLauncher child Launcher should use this._super();
